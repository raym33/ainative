/// ABOUT

This is not another AI wrapper. This is not a gimmick.

We are building the first operating system where AI is not an application layer—it is fundamental to the architecture.

We are assembling the best open source components that already exist, combining them with new code (vibe-coded or handcrafted), and creating something that has never existed before.

> A FUNCTIONAL AI-NATIVE OS PROTOTYPE

/// TARGET PROJECTS FOR COLLABORATION

We are reaching out to the creators of these projects to join forces and build something greater together:

[ KERNEL / BASE ]

  • REDOX OS - Unix-like microkernel in Rust
  • seL4 - Formally verified microkernel
  • Linux kernel - Modified for AI workloads

[ AI INFERENCE ]

  • llama.cpp - Efficient LLM inference
  • whisper.cpp - Local speech recognition
  • MLC-LLM - Universal LLM deployment
  • DeepSpeech - Speech-to-text engine

[ DISPLAY / UI ]

  • Wayland - Modern display protocol
  • wlroots - Modular Wayland library
  • COSMIC - Rust-based desktop

[ SYSTEM TOOLS ]

  • Nushell - Modern shell
  • Starship - Cross-shell prompt
  • Fish - Smart command line

+ Many more to be identified by the community
